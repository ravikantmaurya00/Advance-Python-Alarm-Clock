# Advance-Python-Alarm-Clock

This repository contains an advanced alarm clock application built using Python. The project provides a sleek GUI for setting alarms with options for selecting custom sounds, adding snooze functionality, and controlling alarms with ease. 

# Advanced Python Alarm Clock ‚è∞  

This is an advanced Python-based alarm clock application featuring a user-friendly graphical interface. The app provides functionalities such as alarm scheduling, snoozing, and stopping alarms, along with support for custom alarm sounds.  

## Features  
- **Set Alarm**: Choose the exact time for your alarm with intuitive dropdown menus for hours, minutes, and seconds.  
- **Custom Alarm Sounds**: Select your favorite `.mp3` or `.wav` file to personalize your alarm tone.  
- **Snooze Functionality**: Delay the alarm by a predefined snooze duration (default: 5 minutes).  
- **Stop Alarm**: Disable the alarm when it rings.  
- **Modern GUI**: Built with `tkinter`, the application features a clean and responsive user interface.  
- **Real-Time Clock**: Displays the current time updated every second.
